# 📦 SmartStock
Gestão de inventário inteligente que utiliza análise de dados para prever necessidades de reposição e evitar ruturas de stock.

## ✨ Funcionalidades
- **Alertas Automáticos:** Notificações de stock baixo e movimentações suspeitas.
- **Análise Preditiva:** Projeções de procura baseadas no histórico.
- **Relatórios:** Exportação de dados para Excel e PDF.

## 🛠️ Tecnologias
- **Framework:** TanStack Start
- **Testes:** Playwright (E2E)
- **Deployment:** Cloudflare Wrangler
